# Agent Guide

This repo is a lightweight TanStack Start app being repurposed for a new product direction, so prefer neutral product language unless the task defines a specific direction.

- Package manager: `pnpm`
- Use the Node.js version in `.nvmrc`.
- If Node.js or `pnpm` is missing or on the wrong version, run `scripts/setup-node.sh`.
- Typecheck with `pnpm run lint:types`.
