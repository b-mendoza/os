# Build And Quality Commands

## Core Commands

- `pnpm run dev` тАФ Start development server
- `pnpm run build` тАФ Build for production
- `pnpm run test` тАФ Run all tests
- `pnpm run lint` тАФ Run all lint checks (`oxfmt`, `tsc`, `eslint`)
- `pnpm run fix` тАФ Auto-fix all issues

## Formatting Notes

- This project uses `oxfmt` (not Prettier).
- Format a specific file with `pnpm oxfmt <file>`.
